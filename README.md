# PaddleSlim_demo
PaddleSlim 文档中的 Demo 代码，存在不少 Bug，附上本地环境用于复现

本地 Python 环境：

```
paddlehub                  1.8.2
paddlelite                 2.8
paddlepaddle               2.0.2
paddlepaddle-gpu           2.1.1
paddleseg                  2.1.0
paddleslim                 2.1.0
paddlex                    2.0.0rc4
```

本地硬件环境：
- GPU: TiTAN RTX
- Driver API Version: 11.2, Runtime API Version: 10.2
- cuDNN Version: 7.6.
